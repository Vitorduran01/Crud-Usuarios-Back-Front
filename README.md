# CRUD de Usuários

Projeto CRUD completo desenvolvido com HTML, CSS, JavaScript, Node.js, Express e SQLite.

## Funcionalidades
- Criar usuários
- Listar usuários
- Editar usuários
- Excluir usuários

## Tecnologias
- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQLite

## Como executar
```bash
cd backend
npm install
node index.js
